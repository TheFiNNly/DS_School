Реализация проекта по предсказанию цен на недвижимость. 

EDA - обработка и подготовка данных
	1. EDA_step1 (обработка данных)
	2. EDA_step2 (численные признаки)
	3. EDA_step3 (категориальные признаки)
	4. data_sub_area (подготовка данных для предсказания округа)

Создание моделей
	1. Base_model (базовый вариант модели предсказания цены)
	2. Upgrade_model (допиливание данных)
	3. final_models (сохранение финальных моделей регрессии - предсказания цены)
	4. class_models (модели предсказания округа и района)

Визуализация
	1. area_visual (создание интерактивной карты Москвы с выводом средней стоимости квартиры в районе)
